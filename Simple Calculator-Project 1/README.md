## 🧮 Simple Calculator - Project 1

## 📌 Project Overview
A **GUI-based Calculator** built using **Python and Tkinter**
that performs basic arithmetic operations with a clean
and interactive graphical interface.

---

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Library:** Tkinter (Built-in Python GUI Library)
- **Type:** GUI Application (Graphical User Interface)

---

## ⚙️ Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📐 Modulo ( % )
- 🔢 Decimal point support
- 🔣 Bracket support ( ) for complex expressions
- 🧹 Clear button (C) to reset
- ⚡ Real-time expression evaluation using Python's eval()
- ❌ Error handling for invalid expressions
- 🖥️ Clean GUI window (350x500) with Ridge-style buttons

## 🚀 How to Run

### Prerequisites
- Python 3.x must be installed
- Tkinter comes **pre-installed** with Python (no extra install needed)

### Steps

1. Clone this repository:
```bash
   git clone https://github.com/av648228-cpu/CODSOFT.git
```

2. Navigate to the project folder:
```bash
   cd "CODSOFT/Simple Calculator-Project 1"
```

3. Run the program:
```bash
   python calculator.py
```

4. The Calculator window will open — start calculating! 🎯

---

## 💡 How It Works

- **get_data(value)** → Appends button value to expression string
- **equal_data()** → Evaluates the full expression using `eval()`
- **clear()** → Resets the display and expression string
- All buttons are placed using Tkinter's `.place()` method
- Window is **non-resizable** for consistent layout
